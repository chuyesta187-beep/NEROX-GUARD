# 🛡️ NEROX GUARD

Sistema avanzado de seguridad y automoderación para Discord.

## Características
- Anti-Nuke
- Anti-Raid
- Anti-Link
- Anti-Invite
- Anti-Spam
- Anti-Bot
- Anti-Webhook
- Sistema de Backups
- Logs avanzados
- AutoMod integrado

## Instalación
npm install

## Iniciar el bot
npm start
